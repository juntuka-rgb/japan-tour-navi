**🚲 日本一周NAVI - ルートビルダー
このアプリは、自転車で日本一周を目指す私のルート設定をアシストすることを目的に作成したツールです。

##【このアプリの目的】
* 「なるだけ海沿いを走る」 ルートを快適に計画すること。
* 指定した経由地を通りつつ、「本日の走行予定距離（km）」に入力した距離を移動した地点を特定すること。
* 複雑な経由地を指定した際に発生しやすい「V字戻り（一度来た道を戻る挙動）」を物理的に防ぐロジックを搭載しています。

##【使い方】
* サイドバーにある「出発地」、「本日の走行予定距離（km）」、「最終目的地方面」は必ず入れてください。
* サイドバーにある「今日の⚪︎km地点を計算」をクリックしてください。
* 地図が表示され、自転車アイコンの青いピンが、「計算結果の地点」です。
* 「🚀マップでナビ」をクリックしてください。
* Googleマップが開き、「出発地」から「計算結果の地点」までのルート検索が表示されます。
* ルート検索は、Googleマップで自転車移動のデータがあるエリアは、自転車移動ルートが表示されます。
自転車移動データがないエリアでは、ルートは表示されません。移動を「自動車」や「徒歩」を選択してください。
* 「最終目的地方面」が、「出発地」から「本日の走行予定距離（km）」よりも近い場合は、「最終目的地方面」が「計算結果の地点」として表示されます。
* 岬や半島など、入り組んだエリアで、ショートカットしたくない場合は、経由地を入力してください。

##【ご注意】
* 本アプリは海沿いルートの走行を前提に調整しています。
* 出発地や目的地に 内陸（例：富士山、山間部など）を指定した場合は想定外 です。その場合、期待通りのルートや計算結果にならない可能性がありますので、あらかじめご了承ください。**
